# Neighborhood Farming Engine — Reference

## Use This Reference When Running the Skill
Read this file before creating the deliverable. Use it as a compact checklist and language library, not as text to copy word-for-word.

## 12-Week Farming Theme Bank
1. Local market snapshot.
2. Recent sale story.
3. Home-value review invitation.
4. Seller prep checklist.
5. Buyer demand note.
6. Home maintenance reminder.
7. Pricing myth versus reality.
8. Behind-the-scenes listing launch story.
9. Open-house or local event invitation.
10. Equity check-in.
11. Referral ask tied to the neighborhood.
12. Quarter-end market recap.

## Door-Knock Structure
- Open with a local reason for the visit.
- Ask a low-pressure opinion question.
- Offer a useful one-page resource.
- Ask permission to follow up.
- Record the next step immediately.

## Local Data Guardrails
Use only market facts supplied by the agent or visibly labeled as a field the agent must complete before use. Good data points include sale price, list-to-sale ratio, days on market, inventory count, number of active listings, and recent sale count. Avoid school ratings, crime claims, demographic patterns, or lifestyle assumptions about residents.


## Final Quality Check
- The deliverable is a finished visual file, not a raw chat response.
- The client can send, print, or present it without another formatting step.
- The language sounds like a helpful real estate professional, not a generic marketer.
- The file avoids protected-class language, unsupported claims, pressure tactics, and promises of results.
- Every recommended next step is practical, dated where useful, and tied to the lead's situation.
