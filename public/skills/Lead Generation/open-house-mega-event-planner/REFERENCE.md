# Open House Mega-Event Planner — Reference

## Use This Reference When Running the Skill
Read this file before creating the deliverable. Use it as a compact checklist and language library, not as text to copy word-for-word.

## Mega-Event Timeline
| Timing | Focus |
|---|---|
| 14 days before | Strategy, seller approval, vendor alignment, lead capture setup. |
| 10 days before | Neighbor invite plan, social assets, email announcement. |
| 7 days before | Door-knock route, postcard or flyer drop, signage plan. |
| 3 days before | Reminder posts, partner reminders, talking points. |
| 1 day before | Property prep, sign-in test, route confirmation. |
| Event day | Signage, greeting, qualification, notes, safety, recap. |
| 1–3 days after | Follow-up sequences and seller report. |

## Success Metrics
Visitor count, qualified buyer conversations, seller-neighbor conversations, follow-up appointments, agent-to-agent feedback, social engagement, and post-event listing insights.

## Door-Knock Invite Formula
Local reason, event invitation, quick property detail, no-pressure invite, simple question about neighborhood market interest.


## Final Quality Check
- The deliverable is a finished visual file, not a raw chat response.
- The client can send, print, or present it without another formatting step.
- The language sounds like a helpful real estate professional, not a generic marketer.
- The file avoids protected-class language, unsupported claims, pressure tactics, and promises of results.
- Every recommended next step is practical, dated where useful, and tied to the lead's situation.
