# FSBO Conversion Kit — Reference

## Use This Reference When Running the Skill
Read this file before creating the deliverable. Use it as a compact checklist and language library, not as text to copy word-for-word.

## FSBO Conversation Arc
1. Respect the seller’s decision to try selling independently.
2. Offer help on a specific problem: pricing confidence, buyer screening, safety, negotiation, paperwork coordination, or showing follow-up.
3. Ask discovery questions before pitching.
4. Position the agent as a useful backup plan.
5. Ask for a short strategy appointment only after giving value.

## Objection Map
| Objection | Response Direction |
|---|---|
| I do not want to pay commission | Agree that net matters, then offer to compare net outcomes and risk points. |
| I already have interest | Ask how buyers are being verified and followed up with. |
| I can market it myself | Acknowledge effort, then discuss buyer reach, follow-up, and negotiation management. |
| I had a bad agent before | Validate the concern and explain how the agent’s process is different. |
| I am not ready to list | Offer a light review and a backup plan. |

## Words to Use
- “Would it be helpful if...”
- “Your net matters more than the headline price.”
- “I can give you a second set of eyes.”
- “No pressure to list. This is useful either way.”

## Words to Avoid
- “You will never sell it yourself.”
- “My commission is standard.”
- “I guarantee more money.”
- “You are overpriced.”


## Final Quality Check
- The deliverable is a finished visual file, not a raw chat response.
- The client can send, print, or present it without another formatting step.
- The language sounds like a helpful real estate professional, not a generic marketer.
- The file avoids protected-class language, unsupported claims, pressure tactics, and promises of results.
- Every recommended next step is practical, dated where useful, and tied to the lead's situation.
