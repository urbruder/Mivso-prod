# Cold Database 9-Word Reactivator — Reference

## Use This Reference When Running the Skill
Read this file before creating the deliverable. Use it as a compact checklist and language library, not as text to copy word-for-word.

## 9-Word Message Principles
- Use a simple question.
- Sound like a real person.
- Avoid sales language.
- Ask about current interest, not past failure to respond.
- Make it easy to reply with yes, no, or not now.

## Segment Examples
| Segment | Message Direction |
|---|---|
| Old buyer lead | “Still thinking about buying in Austin this year?” |
| Old seller lead | “Still curious what your home could sell for?” |
| Investor lead | “Still looking for another rental property this year?” |
| Past client | “Still loving the house, or considering changes?” |
| Unknown database contact | “Should I keep sending local market updates?” |

## Response Handling
Positive replies should receive a fast, helpful next step. Negative replies should be acknowledged respectfully. No replies should receive one light follow-up, then move to nurture or inactive status.


## Final Quality Check
- The deliverable is a finished visual file, not a raw chat response.
- The client can send, print, or present it without another formatting step.
- The language sounds like a helpful real estate professional, not a generic marketer.
- The file avoids protected-class language, unsupported claims, pressure tactics, and promises of results.
- Every recommended next step is practical, dated where useful, and tied to the lead's situation.
