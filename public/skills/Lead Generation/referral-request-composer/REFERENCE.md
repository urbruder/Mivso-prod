# Referral Request Composer — Reference

## Use This Reference When Running the Skill
Read this file before creating the deliverable. Use it as a compact checklist and language library, not as text to copy word-for-word.

## Referral Timing Triggers
- Right after a successful closing.
- After a client leaves a review.
- Home anniversary or annual equity update.
- After solving a difficult issue for a client.
- After a market update that creates a useful reason to check in.
- Around a client life event only if the client shared it directly.

## Ask Strength Levels
| Level | When to Use | Example Direction |
|---|---|---|
| Soft | Casual relationship or first ask | “If someone comes to mind...” |
| Specific | Warm relationship and clear timing | “Do you know anyone planning a move this year?” |
| Direct | Strong advocate or repeat referrer | “Could you introduce me by text?” |

## Thank-You Follow-Up
Always thank the person whether or not they have a name. If they provide a referral, ask how they prefer the introduction to happen and record the source clearly.


## Final Quality Check
- The deliverable is a finished visual file, not a raw chat response.
- The client can send, print, or present it without another formatting step.
- The language sounds like a helpful real estate professional, not a generic marketer.
- The file avoids protected-class language, unsupported claims, pressure tactics, and promises of results.
- Every recommended next step is practical, dated where useful, and tied to the lead's situation.
