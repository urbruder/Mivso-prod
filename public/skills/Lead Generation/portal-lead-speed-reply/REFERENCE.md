# Portal Lead Speed-Reply — Reference

## Use This Reference When Running the Skill
Read this file before creating the deliverable. Use it as a compact checklist and language library, not as text to copy word-for-word.

## Intent Patterns
| Lead Signal | Likely Intent | First Reply Angle |
|---|---|---|
| “Is it available?” | Wants quick property status | Answer status if known and offer a showing window. |
| “Can I see it?” | Showing request | Confirm availability and ask for preferred times. |
| Saved search or viewed listing | Early interest | Ask what caught their eye and offer similar options. |
| Financing question | Needs payment clarity | Offer lender conversation without giving loan advice. |
| Vague form fill | Unknown intent | Ask one easy question and offer help narrowing options. |

## First-Reply Formula
1. Use the lead’s name if available.
2. Reference the property or request.
3. Give one useful answer or next step.
4. Ask one question.
5. Offer two simple options, such as “today after 4 or tomorrow morning.”

## Follow-Up Windows
- Immediate: text or call within minutes.
- 15 minutes: call attempt if appropriate.
- 2 hours: useful property detail or similar-home offer.
- Same day: check-in with a simple yes/no question.
- Next day: close-the-loop and invite future help.


## Final Quality Check
- The deliverable is a finished visual file, not a raw chat response.
- The client can send, print, or present it without another formatting step.
- The language sounds like a helpful real estate professional, not a generic marketer.
- The file avoids protected-class language, unsupported claims, pressure tactics, and promises of results.
- Every recommended next step is practical, dated where useful, and tied to the lead's situation.
