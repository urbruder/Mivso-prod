# MLS Listing Syndicator Reference

## Channel Checklist
| Channel Type | Examples | Proof Needed |
|---|---|---|
| MLS and brokerage | MLS live status, brokerage site, agent website, IDX display | Status screenshot or live link |
| Social | Instagram, Facebook, LinkedIn, YouTube Shorts, stories | Post link, scheduled post, or screenshot |
| Email | Buyer list, neighborhood list, internal agents, past-client list | Sent email or scheduled campaign record |
| Print | Flyer, brochure, postcard, sign rider | Approved print file or order confirmation |
| Seller update | Launch recap, links, showing instructions | Copy of seller-facing update |

## Compliance Guardrails
- Confirm status language matches MLS rules.
- Use only approved photos and media.
- Do not reference protected-class demographics.
- Avoid school-quality claims unless using compliant, sourced, neutral language approved by brokerage policy.
- Include required brokerage, license, and MLS attribution where applicable.
