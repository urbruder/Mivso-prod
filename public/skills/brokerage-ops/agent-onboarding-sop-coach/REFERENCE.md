# Agent Onboarding & SOP Coach Reference

## Onboarding Phases
| Phase | Purpose | Typical Items |
|---|---|---|
| Pre-start | Prepare access and expectations before day one | License transfer status, email, calendar, office invite, mentor assignment |
| Day one | Make the agent operational and welcomed | Welcome call, office tour, account logins, brand assets, compliance acknowledgments |
| Week one | Teach the operating rhythm | CRM basics, lead handling, MLS search, showing process, offer review flow |
| Days 8-30 | Turn training into real production behavior | Database setup, buyer consult practice, listing prep shadowing, open house plan |
| Days 31-90 | Move from guided to accountable | Pipeline review, transaction roleplay, lead conversion goals, compliance quality review |

## Core SOP Topics
- Lead intake and response standards
- Buyer consultation process
- Listing appointment prep
- Showing request and safety procedure
- Offer drafting and broker review flow
- Transaction folder naming and document storage
- Communication cadence with clients and internal staff
- Escalation rules for legal, ethical, safety, and compensation questions

## Quality Checks
A complete onboarding packet should answer: who trains the agent, what gets done first, what systems matter, how progress is measured, and when the broker gets involved.
