# One-Click Agent Setup Reference

## Setup Categories
| Category | Typical Items |
|---|---|
| Identity and licensing | License status, office roster, ICA, W-9, emergency contact, policy acknowledgments |
| Communication | Email, calendar, phone, voicemail, internal chat, distribution lists |
| Transaction systems | E-sign, transaction management, document storage, compliance portal |
| MLS and showing | MLS membership, lockbox, showing scheduler, listing input permission |
| CRM and leads | CRM seat, lead routing, source rules, database import, tags |
| Marketing and brand | Headshot, bio, profile, email signature, website, social links, announcement |
| Training | First-week system training, compliance orientation, lead response, transaction workflow |
| Security | Two-factor authentication, access level, password policy, offboarding fallback |

## Security Notes
- Never store passwords in a workbook.
- Confirm license and MLS membership before enabling client-facing tools.
- Use least-privilege access where possible.
- Assign one owner for each setup category.
