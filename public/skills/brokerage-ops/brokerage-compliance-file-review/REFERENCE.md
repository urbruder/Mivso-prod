# Brokerage Compliance File Review Reference

## Default Document Categories
| Category | Examples | Common Review Questions |
|---|---|---|
| Agency and representation | Agency disclosure, buyer-broker agreement, listing agreement | Is it signed, dated, and consistent with the transaction side? |
| Contract and addenda | Purchase agreement, counteroffers, inspection, appraisal, financing, HOA, contingency addenda | Are all pages included and all changes acknowledged? |
| Disclosures | Seller disclosure, lead-based paint, property condition, affiliated business, wire-fraud notice | Is the required disclosure present and signed by required parties? |
| Money and compensation | Commission instructions, referral agreement, earnest-money receipt, DA worksheet | Are amounts, payees, and broker approvals consistent? |
| Closing and settlement | Title docs, escrow instructions, closing statement, final walkthrough, release forms | Are final items present before file closure? |

## Severity Guide
- Critical: item may block closing, payment, or legal file approval.
- High: item must be fixed before broker approval.
- Medium: item is incomplete but likely fixable without delaying closing.
- Low: formatting, naming, or archive-quality issue.
- Broker Judgment: unclear issue that requires broker or counsel decision.
