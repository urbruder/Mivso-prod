# Brokerage BI Briefing Reference

## Suggested KPI Definitions
| KPI | Definition |
|---|---|
| Pending volume | Total sale price for pending transactions in the reporting period |
| Pending sides | Count of brokerage-controlled transaction sides pending |
| Projected GCI | Gross commission expected before splits and fees |
| Projected company dollar | Estimated brokerage revenue after splits before overhead |
| Fall-through risk | Transactions with financing, inspection, appraisal, title, or deadline concerns |
| Agent activity | Calls, appointments, consultations, listings taken, offers written, or team-defined actions |
| Compliance open items | Missing or incomplete file requirements by transaction |
| Recruiting pipeline | Recruit prospects by stage and likely next action |

## Leadership Action Categories
- Revenue acceleration
- Deal-risk intervention
- Agent coaching
- Compliance cleanup
- Recruiting follow-up
- Data hygiene
