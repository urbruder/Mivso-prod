# Smart Document Filing Assistant Reference

## Default Folder Model
| Folder | Document Types |
|---|---|
| 01 Agency and Representation | Agency disclosure, buyer agreement, listing agreement, compensation agreement |
| 02 Contract | Purchase agreement, counteroffer, acceptance, amendments |
| 03 Disclosures | Seller disclosure, lead-based paint, property condition, affiliated business, wire-fraud notice |
| 04 Contingencies and Addenda | Inspection, appraisal, financing, HOA, sale contingency, repair addenda |
| 05 Money and Escrow | EMD receipt, commission instruction, referral agreement, DA worksheet |
| 06 Closing | Title docs, settlement statement, final walkthrough, closing instructions |
| 07 Communications and Notes | Broker review notes, client approvals, key emails or letters |

## Naming Pattern
Preferred pattern: StreetAddress_DocumentType_Date_Status.
Example: 123-Oak-St_Purchase-Agreement_2026-07-01_Signed.pdf

## Review Needed Reasons
- Document type unclear.
- Signatures or dates unclear.
- File appears to contain multiple documents.
- Duplicate or near-duplicate file name.
- Sensitive personal information appears in the name or visible text.
