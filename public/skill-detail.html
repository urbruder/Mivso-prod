How to connect this skill

Step-by-step guides for every major AI platform and coding agent.

🤖

Claude (claude.ai)

Attach and use skills in any Claude conversation
1

Go to claude.ai and open any conversation.

2

Click the "+" attachment icon or drag and drop files into the chat.

3

Upload SKILL.md as a file attachment.

4

In your message, write:

Use the attached skill file to guide your behavior for this task.
5

Optionally add reference files (e.g. examples.json, prompts.md) as additional attachments for richer context.

6

Start your task — Claude will follow the skill instructions throughout the conversation.

💻

Claude Code

Use skills in your terminal-based Claude sessions
1

Open your project in the terminal.

2

Place SKILL.md in your project root or a /skills subfolder.

3

Run claude to start a session, then reference the skill explicitly:

Follow the instructions in SKILL.md for this task
4

Or add it to your CLAUDE.md file so it's always loaded automatically:

# Persistent Skills
Always follow the instructions in /skills/SKILL.md
5

Claude Code will use the skill automatically for every task in that project.

⚡

OpenAI Codex

Inject the skill into your system prompt via Playground or API
1

Open Codex at platform.openai.com or prepare your API call.

2

Copy the full contents of SKILL.md.

3

Paste it at the top of your system prompt in the Playground or API call:

{ "role": "system", "content": "<paste SKILL.md contents here>" }
4

Add your task as the user message.

5

Codex will follow the skill's instructions for that session.

🔀

OpenRouter

Use skills across any model via Playground or API
1

Go to openrouter.ai and open the Playground (or prepare your API call).

2

In the System Prompt field, paste the full text of SKILL.md.

3

Select your preferred model — the skill works with Claude, GPT-4, Mistral and more.

4

Start your conversation — the skill instructions are active from the first message.

5

For API use:

{
  "model": "anthropic/claude-sonnet-4",
  "messages": [
    { "role": "system", "content": "<paste SKILL.md contents here>" },
    { "role": "user", "content": "Your task here" }
  ]
}
🖱️

Cursor

Load skills via project rules or inline chat references
1

Open your project in Cursor.

2

Place SKILL.md in your project root.

3

Open Cursor Settings → Rules for AI and add:

Always read and follow /SKILL.md before starting any task.
4

Or reference it inline in chat:

Follow SKILL.md and then refactor this function
5

Cursor will include the skill in its context automatically via .cursorrules.

🌊

Windsurf (by Codeium)

Apply skills to the Cascade agent for persistent context
1

Open your project in Windsurf.

2

Place SKILL.md in your project root.

3

In the Cascade chat panel, type:

Read SKILL.md and use it as your guide for this session
4

Or create a .windsurfrules file in your root for persistent application:

Always follow the instructions in SKILL.md for this project.
5

Cascade will apply the skill's instructions to all tasks in the session.
